使用說明

  1. 首先在上方設定您的 Gemini API Key 和 RUNPOD API Key
  2. 可選擇性設定提示詞前綴，讓生成的圖片更符合您的需求
  3. 選擇適合的圖片解析度（解析度越高，生成時間越長）
  4. 可添加最多4個 LoRA 模組來微調圖片風格
  5. 在任何網頁上右鍵點擊圖片，選擇「Copy Style」開始分析
  6. 等待 AI 分析完成後，會自動生成相似風格的新圖片
  7. 生成的圖片會自動下載到您的下載資料夾
